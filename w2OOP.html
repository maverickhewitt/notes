Week 2 Java Programming Notes

1. Java Data Types

  • Java has two main categories of data types:
    • Primitive Data Types: Predefined data types that hold a single value.
    • Non-Primitive (Reference) Data Types: Variables that can contain references to objects.

2. Primitive Data Types

  • Common primitive data types include:
    • byte: 8-bit integer
    • short: 16-bit integer
    • int: 32-bit integer
    • long: 64-bit integer
    • float: 32-bit floating point
    • double: 64-bit floating point
    • char: Single 16-bit Unicode character
    • boolean: true or false

3. Default Values for Java Data Types

    Data Type       Default Value
    -------------------------------
    byte            0
    short           0
    int             0
    long            0L
    float           0.0f
    double          0.0d
    char            '\u0000'
    String (or any object) null
    boolean         false

4. Variable Declaration and Initialization

  • Example of variable declaration and initialization:
    •         public class VariableExample {
            public static void main(String[] args) {
                int month = 2, days = 28;
                System.out.println("Month " + month + " has " + days + " Days.");
            }
        }

5. Floating Point Data Types

  • Java has two floating-point data types:
    • float: Single precision (7 decimal points)
    • double: Double precision (15 decimal points)

6. Example of Floating Point Calculation

    public class Sale {
        public static void main(String[] args) {
            double price = 29.75;
            double tax = 1.76;
            double total = price + tax;
            System.out.println("The price of the item is " + price);
            System.out.println("The tax is " + tax);
            System.out.println("The total is " + total);
        }
    }

7. Arithmetic Operators

  • Java has five arithmetic operators:
    • +: Addition
    • -: Subtraction
    • *: Multiplication
    • /: Division
    • %: Modulus

8. Example of Arithmetic Operations

    public class ArithmeticExample {
        public static void main(String[] args) {
            double basePay = 25;
            double regularHours = 40;
            double overtimePay = 37.5;
            double overtimeHours = 10;

            double regularWages = basePay * regularHours;
            double overtimeWages = overtimePay * overtimeHours;
            double totalWages = regularWages + overtimeWages;

            System.out.println("Wages for this week are $" + totalWages);
        }
    }

9. Relational Operators

  • Common relational operators include:
    • ==: Equal to
    • !=: Not equal to
    • >: Greater than
    • <: Less than
    • >=: Greater than or equal to
    • <=: Less than or equal to

10. Logical Operators

  • Common logical operators include:
    • &&: Logical AND
    • ||: Logical OR
    • !: Logical NOT

11. Example of Logical Operations

    public class LogicalExample {
        public static void main(String[] args) {
            int a = 5;
            int b = 10;
            boolean result = (a < b) && (b > 0);
            System.out.println("Result of logical operation: " + result);
        }
    }

12. Type Casting

  • Implicit Casting (Widening): Automatically done by the compiler.
    •         int i = 100;
        long l = i; // Implicit casting
  • Explicit Casting (Narrowing): Must be done manually.
    •         double d = 100.04;
        int i = (int) d; // Explicit casting
