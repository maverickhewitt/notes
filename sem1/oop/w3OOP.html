Week 3 Java Programming Notes

1. Control Structures

  • Control structures are used to dictate the flow of execution in a program.
  • Types of control structures:
    • Selection Structures: if, if...else, switch
    • Repetition Structures: while, do...while, for

2. Selection Structures

2.1 If Statements

    if (condition) {
        // execute statements
    }

2.2 If-Else Statements

    if (condition) {
        // execute statements if true
    } else {
        // execute statements if false
    }

2.3 Nested If Statements

    if (condition1) {
        if (condition2) {
            // execute statements
        } else {
            // execute alternative statements
        }
    }

2.4 Switch Statement

    switch (expression) {
        case value1:
            // execute statements
            break;
        case value2:
            // execute statements
            break;
        default:
            // execute statements if no case matches
    }

3. Example of If-Else Statement

    public class GradeCalculator {
        public static void main(String[] args) {
            int score = 85;
            if (score >= 90) {
                System.out.println("Grade: A");
            } else if (score >= 80) {
                System.out.println("Grade: B");
            } else {
                System.out.println("Grade: C");
            }
        }
    }

4. Repetition Structures

4.1 While Loop

    while (condition) {
        // execute statements
    }

4.2 Do-While Loop

    do {
        // execute statements
    } while (condition);

4.3 For Loop

    for (initialization; condition; update) {
        // execute statements
    }

5. Example of While Loop

    public class WhileLoopExample {
        public static void main(String[] args) {
            int count = 1;
            while (count <= 5) {
                System.out.println("Count: " + count);
                count++;
            }
        }
    }

6. Example of For Loop

    public class ForLoopExample {
        public static void main(String[] args) {
            for (int i = 1; i <= 5; i++) {
                System.out.println("Iteration: " + i);
            }
        }
    }

7. Break and Continue Statements

  • Break Statement: Exits the loop immediately.
    •         while (condition) {
            if (someCondition) {
                break; // exit loop
            }
        }
  • Continue Statement: Skips the current iteration and continues with the next iteration.
    •         for (int i = 0; i < 10; i++) {
            if (i % 2 == 0) {
                continue; // skip even numbers
            }
            System.out.println(i);
        }

8. Example of Break and Continue

    public class BreakContinueExample {
        public static void main(String[] args) {
            for (int i = 1; i <= 10; i++) {
                if (i == 5) {
                    break; // exit loop when i is 5
                }
                System.out.println(i);
            }

            for (int i = 1; i <= 10; i++) {
                if (i % 2 == 0) {
                    continue; // skip even numbers
                }
                System.out.println(i);
            }
        }
    }
