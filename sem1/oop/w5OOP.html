Week 5 Java Programming Notes

1. Methods in Java

  • A method is a collection of statements that perform a specific task.
  • Methods help in code reusability and organization.

2. Defining a Method

    returnType methodName(parameterType parameterName) {
        // method body
    }

3. Calling a Method

  • To call a method, use its name followed by parentheses.
    •         methodName(arguments);

4. Example of a Simple Method

    public class SimpleMethod {
        public static void main(String[] args) {
            greet(); // Calling the method
        }

        public static void greet() {
            System.out.println("Hello, welcome to Java!");
        }
    }

5. Methods with Return Values

  • Methods can return a value using the return statement.
    public class ReturnValueExample {
        public static void main(String[] args) {
            int sum = add(5, 10);
            System.out.println("Sum: " + sum);
        }

        public static int add(int a, int b) {
            return a + b; // Returning the sum
        }
    }

6. Passing Arguments to Methods

  • Arguments can be passed by value.
  • Changes made to parameters do not affect the original arguments.
    public class PassByValue {
        public static void main(String[] args) {
            int number = 10;
            modifyValue(number);
            System.out.println("Number after method call: " + number); // Output: 10
        }

        public static void modifyValue(int num) {
            num = 20; // This change does not affect the original variable
        }
    }

7. Method Overloading

  • Method overloading allows multiple methods with the same name but different parameters.
    public class OverloadingExample {
        public static void main(String[] args) {
            System.out.println("Sum of two integers: " + add(5, 10));
            System.out.println("Sum of three integers: " + add(5, 10, 15));
        }

        public static int add(int a, int b) {
            return a + b;
        }

        public static int add(int a, int b, int c) {
            return a + b + c;
        }
    }

8. Scope of Variables

  • Variables defined inside a method are local to that method.
  • Local variables cannot be accessed outside the method.
    public class VariableScope {
        public static void main(String[] args) {
            int localVar = 5; // Local variable
            System.out.println("Local Variable: " + localVar);
            // System.out.println(globalVar); // This would cause an error
        }
    }

9. Documentation Comments

  • Use @param and @return tags to document methods.
    /**
     * Adds two integers.
     * @param a First integer
     * @param b Second integer
     * @return Sum of a and b
     */
    public static int add(int a, int b) {
        return a + b;
    }

10. Example of a Complete Program

    public class Calculator {
        public static void main(String[] args) {
            int result = add(10, 20);
            System.out.println("Result: " + result);
        }

        public static int add(int a, int b) {
            return a + b;
        }
    }
