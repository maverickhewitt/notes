Week 10 Java Programming Notes

1. Introduction to I/O Streams

  • Streams are a flow of data from a source to a destination.
  • Java supports two types of data in streams:
    • Raw Bytes: Handled by InputStream and OutputStream.
    • Unicode Characters: Handled by Reader and Writer.

2. InputStream and OutputStream

2.1 InputStream Methods

    public class InputStreamExample {
        public static void main(String[] args) throws IOException {
            InputStream input = System.in; // Standard input stream
            int data = input.read(); // Read a byte of data
            System.out.println("You entered: " + (char)data);
        }
    }

2.2 OutputStream Methods

    public class OutputStreamExample {
        public static void main(String[] args) throws IOException {
            OutputStream output = System.out; // Standard output stream
            output.write('H'); // Write a single byte
            output.write('i'); // Write another byte
            output.flush(); // Flush the output stream
        }
    }

3. Reading and Writing Files with Byte Streams

3.1 Reading from a File

    public class FileReadExample {
        public static void main(String[] args) {
            try (FileInputStream fin = new FileInputStream("input.txt")) {
                int i;
                while ((i = fin.read()) != -1) {
                    System.out.print((char) i); // Print each character
                }
            } catch (IOException e) {
                System.out.println("File not found: " + e.getMessage());
            }
        }
    }

3.2 Writing to a File

    public class FileWriteExample {
        public static void main(String[] args) {
            try (FileOutputStream fout = new FileOutputStream("output.txt")) {
                String data = "Hello, World!";
                fout.write(data.getBytes()); // Write string data to file
            } catch (IOException e) {
                System.out.println("Error writing to file: " + e.getMessage());
            }
        }
    }

4. Reader and Writer Classes

4.1 Using Reader

    public class FileReaderExample {
        public static void main(String[] args) {
            try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
                String line;
                while ((line = reader.readLine()) != null) {
                    System.out.println(line); // Print each line
                }
            } catch (IOException e) {
                System.out.println("Error reading file: " + e.getMessage());
            }
        }
    }

4.2 Using Writer

    public class FileWriterExample {
        public static void main(String[] args) {
            try (BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"))) {
                writer.write("Hello, World!"); // Write a line to the file
                writer.newLine(); // Add a new line
                writer.write("This is a second line.");
            } catch (IOException e) {
                System.out.println("Error writing to file: " + e.getMessage());
            }
        }
    }

5. Using Scanner for Input

    import java.util.Scanner;

    public class ScannerExample {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            System.out.print("Enter your name: ");
            String name = scanner.nextLine(); // Read a line of text
            System.out.println("Hello, " + name + "!");
            scanner.close(); // Close the scanner
        }
    }

6. Summary of Key Concepts

  • Java provides a variety of classes for handling input and output operations.
  • Input and output can be performed using byte streams or character streams.
  • Files can be read and written using FileInputStream and FileOutputStream.
  • Using Scanner simplifies reading input from the console.
