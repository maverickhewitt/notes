Java Programming Notes

1. Java Program Structure

  • A Java program is composed of one or more classes.
  • The syntax for declaring a class:
    • class <class name> {
    <class member declarations>
}

2. Main Class and Main Method

  • One class must be designated as the main class.
  • The main class must define a method called main:
    • public static void main(String[] args) {
    // Code to be executed
}

3. Comments in Java

  • Comments are used to explain code and are ignored by the compiler.
  • Single-line comment: // This is a comment
  • Multi-line comment:
    • /*
This is a 
multi-line comment
*/

4. Import Statements

  • Classes in the Java API are organized into packages.
  • Explicit import:
    • import java.util.Scanner;
  • Wildcard import:
    • import java.util.*;

5. Console Output

  • Using System.out.println for output:
    • System.out.println("Welcome to Java Programming!");
  • Using printf for formatted output:
    • System.out.printf("The value is %d", value);

6. Variables and Data Types

  • Variable declaration:
    • datatype identifier;
int num1;
  • Variable initialization:
    • boolean result = true;
char capitalC = 'C';
  • Example of variable usage:
    • public class Variable {
    public static void main(String[] args) {
        int value;
        value = 5;
        System.out.print("The value is ");
        System.out.println(value);
    }
}

7. String Concatenation

  • Joining strings and other data types:
    • System.out.println("The total is " + total);

8. Primitive Data Types

  • Common primitive data types:
    • int: Integer values
    • double: Floating-point values
    • boolean: true/false values
    • char: Single character

9. Example of a Simple Java Program

public class Welcome {
    public static void main(String[] args) {
        System.out.println("Welcome to Java Programming!");
    }
}

10. Error Types in Java

  • Syntax Errors: Detected during compilation.
  • Run-Time Errors: Detected during execution.
  • Logic Errors: Errors in the algorithm.

11. Identifiers

  • Rules for identifiers:
    • Can contain letters, digits, underscores, and dollar signs.
    • Cannot start with a digit.
    • Case-sensitive (e.g., myVar is different from myvar).

12. Example of Variable Declaration and Initialization

public class Initialize {
    public static void main(String[] args) {
        int month = 2, days = 28;
        System.out.println("Month " + month + " has " + days + " Days.");
    }
}

13. Escape Sequences

  • Used for special characters:
    • \n: New line
    • \t: Tab
    • \\: Backslash
