Chapter 2: Sensors, Actuators, and Microcontrollers

2.1 Learn Electronics

2.1.1 Basic Electronic Terminology & Concepts

  • Electronics Definition: Study of controlling electricity using physical components and circuits.
  • Core Concepts:
    • Electric Current: Flow of electrons (measured in Amperes, A).
    • Voltage (V): Electrical potential difference (e.g., battery voltage).
    • Resistance (Ω): Opposition to current flow (e.g., resistors).
    • Power (W): Rate of energy transfer (\( P = V \times I \)).
  • Ohm's Law:
    • Formula: \( V = I \times R \).
    • Example: A 6V battery with a 2.2kΩ resistor limits current to protect an LED.
  • Basic Circuit Components:
    • Battery: Energy source (e.g., 6V DC).
    • Resistor: Limits current (e.g., 2.2kΩ for LED protection).
    • LED: Light-emitting diode (symbol includes arrows for light emission).

2.1.2 Advanced Electronic Terminology

  • Circuit Types:
    • Series: Components connected end-to-end (same current).
    • Parallel: Components share common nodes (same voltage).
  • Component Categories:
    • Active: Require external power (e.g., transistors).
    • Passive: No external power (e.g., resistors, capacitors).
  • Current Types:
    • DC (Direct Current): Unidirectional flow (e.g., batteries, solar cells).
    • AC (Alternating Current): Bidirectional flow (e.g., power grids).
  • Key Components:
    • Solenoids: Electromagnetic actuators (e.g., door latches, valves).
    • Relays: Electrically controlled switches (e.g., high-voltage control).

2.1.3 From Schematic to PCB

  • Design Phase:
    • Concept: Define circuit purpose.
    • Research: Select components (e.g., sensors, resistors).
    • Schematic: Standardized diagram (e.g., symbols for LEDs, resistors).
  • Prototype Phase:
    • Breadboard: Solderless prototyping tool.
    • PCB Layout: Design for printed circuit boards.
    • Testing: Validate functionality (e.g., voltage checks).
  • Production Phase:
    • PCB Manufacturing: Mass production of circuits.
    • Maintenance: Firmware updates, component replacements.

2.2 Microcontrollers: SparkFun Inventor’s Kit (SIK)

2.2.1 Kit Components

  • Hardware:
    • SparkFun RedBoard (Arduino-compatible).
    • Solderless breadboard, jumper wires, USB cable.
    • Resistors (e.g., 330Ω, 10kΩ), LEDs, sensors (temperature, flex).
  • Software:
    • Arduino IDE for programming (.ino sketches).
    • Key functions: setup(), loop(), digitalWrite().

2.2.2 Simple Circuits & Programming

  • Example Circuit:
    • Connect LED + resistor in series to RedBoard pins.
    • Code snippet: 
      void setup() { pinMode(13, OUTPUT); }
void loop() { digitalWrite(13, HIGH); }
  • Debugging:
    • Compile with ✓ (checkmark) in Arduino IDE.
    • Upload code with → (arrow) icon.

2.2.3 Sensors & Actuators

  • Sensors in SIK:
    • Soft Potentiometer: Pressure-sensitive resistor.
    • Photo Resistor: Light-dependent resistance.
    • Temperature Sensor: Outputs analog voltage proportional to temperature.
  • Actuators:
    • Servo Motor: Precise angular control.
    • Relay Module: Switches high-power devices (e.g., lamps).

2.3 Packet Tracer 7.x and IoT

  • IoT Simulation:
    • Model end-to-end systems (e.g., smart home, industrial IoT).
    • Components Group: MCU (Arduino-like), SBC (Raspberry Pi-like), sensors/actuators.
  • Key Features:
    • IoE Custom Cable: Connects IoT devices to MCU/SBC.
