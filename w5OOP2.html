Week 5 Part II Java Programming Notes

1. Static Methods and Fields

  • Static methods belong to the class rather than any specific instance.
  • Static fields are shared among all instances of a class.

2. Defining Static Methods

    public class Example {
        public static void displayMessage() {
            System.out.println("This is a static method.");
        }
    }

3. Calling Static Methods

    public class Main {
        public static void main(String[] args) {
            Example.displayMessage(); // Calling static method
        }
    }

4. Static Variables Example

    public class Counter {
        static int count = 0; // Static variable

        public Counter() {
            count++; // Increment count for each instance
        }

        public static void displayCount() {
            System.out.println("Count: " + count);
        }
    }

5. Example of Static Variable Usage

    public class Main {
        public static void main(String[] args) {
            new Counter();
            new Counter();
            Counter.displayCount(); // Output: Count: 2
        }
    }

6. Enumerated Types (Enums)

  • Enums are a special data type that enables a variable to be a set of predefined constants.
  • Syntax for defining an enum:
    •         enum Day {
            SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY
        }

7. Using Enums in Java

    public class EnumExample {
        enum Day { SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY }

        public static void main(String[] args) {
            Day today = Day.WEDNESDAY;
            System.out.println("Today is: " + today);
        }
    }

8. Enum Methods

  • toString(): Returns the name of the enum constant.
  • ordinal(): Returns the zero-based position of the constant in the enum.
    public class EnumMethods {
        enum Day { SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY }

        public static void main(String[] args) {
            Day today = Day.FRIDAY;
            System.out.println("Today is: " + today.toString());
            System.out.println("Ordinal of today: " + today.ordinal());
        }
    }

9. Using Enums in Switch Statements

    public class EnumSwitch {
        enum Day { SUNDAY, MONDAY, TUESDAY, WEDNESDAY, THURSDAY, FRIDAY, SATURDAY }

        public static void main(String[] args) {
            Day today = Day.MONDAY;

            switch (today) {
                case SUNDAY:
                    System.out.println("It's Sunday!");
                    break;
                case MONDAY:
                    System.out.println("It's Monday!");
                    break;
                default:
                    System.out.println("It's another day.");
            }
        }
    }

10. Summary of Key Concepts

  • Static methods and fields are associated with the class itself.
  • Enums provide a way to define a set of named constants.
  • Enums can be used in switch statements for cleaner code.
