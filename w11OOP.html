Week 11 Java Programming Notes

1. Introduction to GUI Programming

  • GUI programming involves creating visual interfaces for users to interact with.
  • Java provides several libraries for GUI programming, including AWT and Swing.

2. Creating a GUI Window

    import javax.swing.*;

    public class GUIWindow {
        public static void main(String[] args) {
            JFrame frame = new JFrame("My First GUI Window");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);
            frame.setVisible(true);
        }
    }

3. Adding Components to a GUI Window

    import javax.swing.*;

    public class GUIComponents {
        public static void main(String[] args) {
            JFrame frame = new JFrame("My First GUI Window");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);

            JLabel label = new JLabel("Hello, World!");
            frame.add(label);

            frame.setVisible(true);
        }
    }

4. Handling Events in a GUI Window

    import javax.swing.*;
    import java.awt.event.*;

    public class GUIEvents {
        public static void main(String[] args) {
            JFrame frame = new JFrame("My First GUI Window");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);

            JButton button = new JButton("Click Me!");
            button.addActionListener(new ActionListener() {
                public void actionPerformed(ActionEvent e) {
                    System.out.println("Button clicked!");
                }
            });
            frame.add(button);

            frame.setVisible(true);
        }
    }

5. Using Layout Managers in a GUI Window

    import javax.swing.*;
    import java.awt.*;

    public class GUILayout {
        public static void main(String[] args) {
            JFrame frame = new JFrame("My First GUI Window");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);

            JPanel panel = new JPanel();
            panel.setLayout(new FlowLayout());

            JButton button1 = new JButton("Button 1");
            JButton button2 = new JButton("Button 2");
            JButton button3 = new JButton("Button 3");

            panel.add(button1);
            panel.add(button2);
            panel.add(button3);

            frame.add(panel);

            frame.setVisible(true);
        }
    }

6. Creating a GUI Menu

    import javax.swing.*;
    import java.awt.*;

    public class GUIMenu {
        public static void main(String[] args) {
            JFrame frame = new JFrame("My First GUI Window");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setSize(300, 200);

            JMenuBar menuBar = new JMenuBar();
            frame.setJMenuBar(menuBar);

            JMenu fileMenu = new JMenu("File");
            menuBar.add(fileMenu);

            JMenuItem openItem = new JMenuItem("Open");
            fileMenu.add(openItem);

            JMenuItem saveItem = new JMenuItem("Save");
            fileMenu.add(saveItem);

            frame.setVisible(true);
        }
    }

7. Summary of Key Concepts

  • GUI programming involves creating visual interfaces for users to interact with.
  • Java provides several libraries for GUI programming, including AWT and Swing.
  • GUI windows can be created using the JFrame class.
  • Components can be added to a GUI window using the add method.
  • Events can be handled in a GUI window using event listeners.
  • Layout managers can be used to arrange components in a GUI window.
  • GUI menus can be created using the JMenuBar and JMenu classes.
