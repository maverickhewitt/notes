Week 6 Java Programming Notes

1. Introduction to Arrays

  • An array is a sequenced collection of elements of the same data type.
  • Arrays are used to store multiple values in a single variable.

2. Declaring and Creating Arrays

    // Declaration
    int[] numbers; // or int numbers[];

    // Creation
    numbers = new int[5]; // Array of 5 integers

3. Initializing Arrays

  • Two ways to initialize an array:
    •         // Method 1: Separate declaration and initialization
        int[] numbers = new int[3];
        numbers[0] = 1;
        numbers[1] = 2;
        numbers[2] = 3;

        // Method 2: Inline initialization
        int[] numbers = {1, 2, 3};

4. Accessing Array Elements

    public class ArrayAccess {
        public static void main(String[] args) {
            int[] numbers = {10, 20, 30, 40, 50};
            System.out.println("First element: " + numbers[0]); // Output: 10
        }
    }

5. Looping Through Arrays

    public class ArrayLoop {
        public static void main(String[] args) {
            int[] numbers = {1, 2, 3, 4, 5};
            for (int i = 0; i < numbers.length; i++) {
                System.out.println("Element at index " + i + ": " + numbers[i]);
            }
        }
    }

6. Enhanced For Loop

    public class EnhancedForLoop {
        public static void main(String[] args) {
            String[] names = {"Alice", "Bob", "Charlie"};
            for (String name : names) {
                System.out.println("Name: " + name);
            }
        }
    }

7. Multi-Dimensional Arrays

  • Multi-dimensional arrays are arrays of arrays.
    •         // Declaration and creation of a 2D array
        int[][] matrix = new int[3][3]; // 3x3 matrix

8. Accessing Multi-Dimensional Array Elements

    public class MultiDimensionalArray {
        public static void main(String[] args) {
            int[][] matrix = {
                {1, 2, 3},
                {4, 5, 6},
                {7, 8, 9}
            };
            System.out.println("Element at (1, 2): " + matrix[1][2]); // Output: 6
        }
    }

9. Strings in Java

  • Strings are objects that represent a sequence of characters.
  • Strings are immutable, meaning they cannot be changed once created.

10. Creating and Initializing Strings

    public class StringExample {
        public static void main(String[] args) {
            String greeting = "Hello, World!";
            System.out.println(greeting);
        }
    }

11. Common String Methods

  • length(): Returns the length of the string.
  • charAt(index): Returns the character at the specified index.
  • substring(start, end): Returns a substring from start to end index.
  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.

12. Example of String Methods

    public class StringMethods {
        public static void main(String[] args) {
            String text = "Java Programming";
            System.out.println("Length: " + text.length()); // Output: 16
            System.out.println("Character at index 0: " + text.charAt(0)); // Output: J
            System.out.println("Substring: " + text.substring(0, 4)); // Output: Java
            System.out.println("Uppercase: " + text.toUpperCase()); // Output: JAVA PROGRAMMING
        }
    }

13. String Concatenation

    public class StringConcatenation {
        public static void main(String[] args) {
            String firstName = "John";
            String lastName = "Doe";
            String fullName = firstName + " " + lastName;
            System.out.println("Full Name: " + fullName); // Output: John Doe
        }
    }

14. String Comparison

  • Use equals() to compare strings for content equality.
  • Use == to compare string references.
    public class StringComparison {
        public static void main(String[] args) {
            String str1 = new String("Hello");
            String str2 = new String("Hello");
            System.out.println(str1.equals(str2)); // Output: true
            System.out.println(str1 == str2); // Output: false
        }
    }
